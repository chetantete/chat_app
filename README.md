# Rails Chat Application

This is rails chat application demo
This supports single chat as well as group chat feature using actioncable.


### Installation

```sh
$ rvm 2.5.1
$ bundle
$ update database.yml file
$ rake db:create
$ rake db:migration
$ rake db:seed
$ rails s
```
